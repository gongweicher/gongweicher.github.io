  1. std::function, std::bind

std::function是一个函数包装器，包装任意类型的可调用实体(函数、函数指针、函数对象、lamda)，可拷贝，可移动。定义格式：std::function<函数类型>,例typedef std::function<void (int,int)> Callback;

std::bind是通用的函数适配器，将可调用对象与其参数绑定在一起,返回类型为函数对象『如果一个类将()运算符重载为成员函数，这个类就称为函数对象类』。

    与std::bind类似，std::thread也必须显式通过std::ref来引用传参，否则形参的引用声明是无效的。

    std::thread thd(t_func, str, i);,the argument str, will be copy ,then forwarded as an rvalue. if the fucntion expects an lvalue reference then it can’t be called with an rvalue.
    The arguments to the thread function are moved or copied by value. If a reference argument needs to be passed to the thread function, it has to be wrapped (e.g., with std::ref or std::cref).
